## Build Summary

v2.3.1-Build3 completes the Image Manager feature set with enhanced admin UX and critical bug fixes.

## Changes from v2.3.1-Build2

### Image Path Routing
- Bare filenames (no path) now route to blob storage
- Fixes 404s for events with `image: "filename.jpg"` format

### Admin Images Tab Enhancements
- **Copy feedback:** Green checkmark + "✓ Copied!" (replaces alert)
- **Real-time search:** Filter images as you type
- **Dynamic count:** Shows "X of Y" when filtering
- **Clear button:** Appears when search is active
- **Preview modal:** Click image to enlarge, click outside to close
- **Auto-load:** Images load when tab is activated (already existed)

### Bug Fixes
- **Location/Venue fix:** New events now set `venue = location` when venue is empty
- Resolves hardcoded "Jay Pritzker Pavilion" appearing in calendar links
