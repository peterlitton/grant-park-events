## ✨ Features

### **1. Index Report Enhancements**
- Improved UI for page management
- Better status indicators
- Enhanced filtering capabilities

### **2. Email Campaign Status Cycling**
- Single button cycles through: Draft → Scheduled → Sent → Draft
- Confirmation prompts for status changes
- Automatic timestamp recording

### **3. Index Report URL Display Changes**
- Column header: "URL HTTPS://WWW.GRANTPARKEVENTS.COM/"
- Event URLs display with "events/" prefix
- Static pages display without leading slash

### **4. Future Events Filter**
- Checkbox: "Future Events Only"
- Default: Unchecked (shows all events)
- When checked: Shows only published events with date >= today
- Combines with existing filters

### **5. Event Date Sorting**
- Events sorted chronologically (earliest first)
- Events with dates appear before non-events
- Consistent sorting across all views

### **6. GA4 Integration Improvements**
- Enhanced event tracking
- Better analytics data collection

### **7. MailerLite Integration**
- Email campaign management
- Subscriber handling
- Campaign tracking
