## ✨ UI Improvements

### **1. Admin Index Report - Arrow Wrapping Fix**

**Issue:** Column headers "Event Date ↓" and "Updated ↓" were wrapping to multiple lines on narrow screens.

**Fix:** Added `whitespace-nowrap` to sortable column headers.

**Result:** 
- Headers stay on single line
- Arrows never wrap below text
- Cleaner, more professional appearance

---

### **2. Email PNG Header - Reduced Vertical Space**

**Issue:** Email header had excessive white space above hero image.

**Changes:**
- **Vertical padding:** 20px → 7px (65% reduction)
- **Logo size:** 200px → 266px width (33% bigger)
- **QR placeholder:** 80×80px → 66×66px (fills tighter space)
- **Visual bar height:** ~120px → ~80px (33% reduction)

**Result:**
- More compact, professional header
- Larger logo presence
- QR code visually fills available space
- Better visual balance

**Files Modified:**
- `admin-index-report.html` - Added whitespace-nowrap to headers
- `netlify/functions/generate-email-html.js` - Updated header dimensions
