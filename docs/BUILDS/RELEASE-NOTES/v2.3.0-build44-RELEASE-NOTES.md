### **Issue #2: Missing Favicon on Event Pages** ✅ FIXED

**Problem:**
- Homepage shows favicon in browser tab ✅
- Event pages missing favicon in browser tab ❌
- Example: `/events/2026-05-27-navy-pier-summer-fireworks-51`

**Root Cause:**
- Static favicon in HTML head
- Dedicated event pages load dynamically
- Favicon link not created for dynamic pages

**Fix:**
```javascript
// Added to event page meta tag update:
let favicon = document.querySelector('link[rel="icon"]');
if (!favicon) {
  favicon = document.createElement('link');
  favicon.rel = 'icon';
  favicon.type = 'image/x-icon';
  favicon.href = '/assets/favicon.ico';
  document.head.appendChild(favicon);
}
```

**Result:**
- Homepage: favicon ✅
- Event pages: favicon ✅
- About/Signup: favicon ✅
- Admin pages: don't need it (as requested)
