7. **ALL YOUR EVENTS** with proper URLs

**Event URLs shown as:**
```
/events/2026-05-24-navy-pier-fireworks-123456
/events/2026-06-12-orchestra-concert-789012
```

**Event Status:**
- **⏳ Pending** - Published, waiting for Google to index
- **🚫 Noindex** - Past event (7+ days old, intentionally not indexed)
- **❌ Excluded** - Draft (not published yet)

---

## 🔧 HOW IT WORKS

**Function pulls your events from Netlify Blobs:**
1. Reads events data
2. Generates URL for each event (date-first format)
3. Calculates if event is past (7+ days)
4. Assigns appropriate status
5. Displays in table

**No manual entry needed - it's automatic!**
