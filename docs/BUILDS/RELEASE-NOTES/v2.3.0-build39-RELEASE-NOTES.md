## 📋 FEATURE 1: PAST EVENT NOINDEX

**User Request:** "Keep past events accessible but add noindex meta tag after event passes, remove from sitemap 7 days after event"

### **Implementation:**

**A. Noindex Meta Tag (Automatic)**

Events 7+ days past automatically get:
```html
<meta name="robots" content="noindex, follow">
```

**Logic:**
```javascript
const eventDate = parseLocalDate(ev.date);
const today = new Date();
const daysPast = Math.floor((today - eventDate) / (1000 * 60 * 60 * 24));

if(daysPast > 7) {
  // Add noindex tag
}
```

**Behavior:**
- Event on 2026-01-15
- Today is 2026-01-29
- Days past: 14
- Result: Noindex applied ✅

**What this means:**
- Google will eventually deindex the page
- Page still accessible via URL
- Link equity preserved ("follow" kept)
- Historical value maintained

---

**B. Sitemap Exclusion**

**Current:** sitemap-events.xml includes all events

**Recommended Update:**

Generate sitemap dynamically to exclude past events:

```javascript
// Only include upcoming + recent events
const includeInSitemap = (event) => {
  const eventDate = new Date(event.date);
  const today = new Date();
  const daysAgo = (today - eventDate) / (1000 * 60 * 60 * 24);
  
  // Include: Upcoming + 7 days past
  return daysAgo <= 7;
};

const sitemapEvents = events.filter(includeInSitemap);
```

**Action Required:**
- Update sitemap generation logic
- Or manually remove past events from sitemap weekly
- Resubmit to Google Search Console

**B. Status Indicators:**
```
✅ Indexed     - Page is in Google index
⏳ Pending     - Submitted, waiting for crawl
🔄 Requested   - Index request sent
🚫 Noindex     - Past event (intentional)
❌ Excluded    - Google excluded (investigate)
```

- Events only

- URL
- Type (index, event, static)
- Status (with icon)
- Details
- Actions

**File:** `netlify/functions/gsc-request-index.js`

```javascript
const { google } = require('googleapis');

exports.handler = async (event, context) => {
  const { url } = JSON.parse(event.body);
  
  try {
    const credentials = JSON.parse(process.env.GSC_CREDENTIALS);
    
    const auth = new google.auth.GoogleAuth({
      credentials,
      scopes: ['https://www.googleapis.com/auth/webmasters']
    });
    
    const indexing = google.indexing({ version: 'v3', auth });
    
    await indexing.urlNotifications.publish({
      requestBody: {
        url: url,
        type: 'URL_UPDATED'
      }
    });
    
    return {
      statusCode: 200,
      body: JSON.stringify({ success: true })
    };
  } catch (error) {
    return {
      statusCode: 500,
      body: JSON.stringify({ error: error.message })
    };
  }
};
```
