## 🔧 FIXES IN BUILD20

### ✅ ISSUE 1: Dedicated Event Pages - Images Now Display

**Problem:** Images not showing on dedicated event pages like:
- `/events/brahms-symphony-no-4-1769615765687`
- `/events/brahms-symphony-no-4-2`

**Root cause:** Relative image paths don't resolve in dedicated page context

**Fix:** Added absolute URL conversion for dedicated event page images

**Result:** ✅ Images display correctly on all dedicated event pages

---

### ✅ ISSUE 2: Admin Preview Cards - Clean Text Snippets

**Problem:** Admin event card snippets showed HTML:
```
<!--StartFragment-->Percussive bombast...&nbsp;<i>Military</i>&nbsp;Symphony
```

**Expected:** Plain text only:
```
Percussive bombast... Military Symphony
```

**Fix:** Added `stripHTML()` function to remove all HTML tags from preview snippets

**Result:** ✅ Admin cards show clean, readable text snippets

---

### ✅ ISSUE 3: Paste Filter - Paragraph Breaks Preserved

**Problem:** Pasting multi-paragraph content collapsed paragraphs

**Fix:** Added `<p>` tag to allowed tags list in paste filter

**Result:** ✅ Paragraph breaks preserved when pasting

**Allowed tags now:**
- `<strong>`, `<b>` - Bold
- `<em>`, `<i>` - Italic
- `<ul>`, `<ol>`, `<li>` - Lists
- `<br>` - Line breaks
- `<p>` - Paragraphs ⭐ NEW

## 📋 CHANGES SUMMARY

### **index.html:**
**Dedicated Event Page Image Fix:**
```javascript
// BEFORE
src: ev.image

// AFTER
src: ev.image.startsWith('http') || ev.image.startsWith('data:') ? 
  ev.image : 
  (window.location.origin + '/' + ev.image)
```

### **admin.html:**
**New stripHTML Function:**
```javascript
const stripHTML = (html) => {
  if (!html) return '';
  const temp = document.createElement('div');
  temp.innerHTML = html;
  return temp.textContent || temp.innerText || '';
};
```

**Preview Card Snippet:**
```javascript
// BEFORE
event.description.substring(0, 150) + '...'

// AFTER
stripHTML(event.description).substring(0, 150) + '...'
```

**Paste Filter Updated:**
```javascript
// BEFORE
const allowedTags = ['STRONG', 'B', 'EM', 'I', 'UL', 'OL', 'LI', 'BR'];

// AFTER
const allowedTags = ['STRONG', 'B', 'EM', 'I', 'UL', 'OL', 'LI', 'BR', 'P'];
```
