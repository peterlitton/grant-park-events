### **3. Calendar Default Month - INTELLIGENT** ✅
**User:** "on desktop calendar view the default month should be the month with the soonest published event. if no published events current month. so i would expect to see May 2026 as the default month"

**Old behavior:**
- Hardcoded to June 2026: `new Date(2026, 5)`
- Always showed June regardless of events

**New behavior:**
```javascript
const getInitialCalendarMonth = (eventsList) => {
  // Filter published events and sort by date
  const publishedEvents = eventsList
    .filter(ev => ev.published && ev.date)
    .sort((a, b) => new Date(a.date) - new Date(b.date));
  
  if (publishedEvents.length === 0) {
    return new Date(); // Current month if no published events
  }
  
  // Return month of first published event
  const firstEventDate = parseLocalDate(publishedEvents[0].date);
  return new Date(firstEventDate.getFullYear(), firstEventDate.getMonth(), 1);
};
```

**Logic:**
1. Gets all published events
2. Sorts by date (earliest first)
3. Returns month of soonest event
4. Falls back to current month if no published events

**Expected Result:**
- If soonest published event is in May 2026 → Shows May 2026 ✅
- If no published events → Shows current month (January 2026) ✅
- Updates dynamically when events load ✅
