## 📊 TECHNICAL PATTERN

**Absolute URL Conversion Pattern:**
```javascript
// Check if already absolute or data URI
if (path.startsWith('http') || path.startsWith('data:')) {
  return path; // Use as-is
} else {
  return window.location.origin + '/' + path; // Convert to absolute
}
```

**Handles:**
- `assets/events/concert.jpg` → `https://grantparkevents.com/assets/events/concert.jpg`
- `https://example.com/image.jpg` → `https://example.com/image.jpg` (unchanged)
- `data:image/jpeg;base64,...` → `data:image/jpeg;base64,...` (unchanged)
