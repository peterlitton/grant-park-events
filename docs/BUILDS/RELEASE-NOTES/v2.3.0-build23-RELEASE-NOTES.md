## 🎯 HOW IT WORKS

**User Flow:**
1. User visits site
2. Wait configured delay (default 10 seconds)
3. Popup appears with email signup
4. User enters email → Subscribes
5. Success message → Auto-close after 2 seconds
6. Cookie set → Won't show again for 30 days

**Admin Control:**
1. Go to Admin → Popup tab
2. Toggle ON/OFF
3. Adjust delay slider
4. Edit headline, description, button text
5. Enable "Testing Mode" to see popup every time

## 📝 TECHNICAL DETAILS

### **Component Structure:**
```javascript
const EmailPopup = () => {
  // State
  const [showPopup, setShowPopup] = useState(false);
  const [email, setEmail] = useState('');
  const [settings, setSettings] = useState(null);
  
  // Load settings & check cookie
  useEffect(() => {
    const popupSettings = localStorage.getItem('gpe_popup_settings');
    const hasSeenPopup = getCookie('gpe_popup_seen');
    
    if (enabled && (!hasSeenPopup || ignoreCookie)) {
      setTimeout(() => setShowPopup(true), delay * 1000);
    }
  }, []);
  
  // Submit handler
  const handleSubmit = async () => {
    await fetch('/.netlify/functions/emailoctopus-subscribe', {
      body: JSON.stringify({ email })
    });
    setCookie('gpe_popup_seen', 'true', 30);
  };
};
```

### **Cookie Format:**
```
gpe_popup_seen=true; expires=...; path=/; SameSite=Lax
```

### **z-index:**
Popup uses `z-[9999]` to ensure it appears above all content.
