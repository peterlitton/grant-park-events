## 📊 TECHNICAL DETAILS

### **Why Modal Images Need Absolute URLs:**

**Context matters:**
- **Card images:** Rendered in normal page flow → Relative paths resolve correctly
- **Modal images:** Rendered in overlay/portal → Need absolute URLs

**The fix:**
```javascript
// Check if path is already absolute
if (selectedEvent.image.startsWith('http')) {
  // Use as-is (absolute URL or data URI)
  return selectedEvent.image;
} else {
  // Convert relative path to absolute
  return window.location.origin + '/' + selectedEvent.image;
}
```

**This handles:**
- `assets/events/image.jpg` → `https://grantparkevents.com/assets/events/image.jpg`
- `https://example.com/image.jpg` → `https://example.com/image.jpg` (unchanged)
- `data:image/jpeg;base64,...` → `data:image/jpeg;base64,...` (unchanged)
