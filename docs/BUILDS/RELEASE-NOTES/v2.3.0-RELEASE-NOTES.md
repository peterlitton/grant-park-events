1. Upload ZIP to Netlify
2. **Add environment variables** (see above)
3. **Trigger new deployment** (critical - variables don't work without redeploy!)
4. Wait 2 minutes
5. Test signup form (Sign Up tab)
6. Test contact form (About tab)

## Troubleshooting

**If EmailOctopus not working:**
- Verify environment variables added
- Verify redeployed after adding variables
- Check browser console (F12) for errors
- Check Netlify Function logs

**If Contact Form not working:**
- Verify Netlify Forms enabled (Site Settings → Forms)
- Check spam folder
- Wait 2-3 minutes for email
- Configure email notifications in Netlify Forms

**See:** FORM-ISSUES-TROUBLESHOOTING.md for complete diagnostic guide
