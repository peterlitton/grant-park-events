📧 Email Campaign Workflow

  1. Create campaign in Admin → Email Campaigns
  2. Click "Preview" to verify content
  3. Click "Copy HTML" to copy to clipboard
  4. In MailerLite: Create Campaign → Custom HTML Editor → Paste
  5. Enable "Google Analytics link tracking" before sending
  6. Send test email to yourself, then schedule/send
  7. Back in Admin: Click "Mark Sent" to update status
